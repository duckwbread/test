The Lore

The first reports surfaced after the 2018 Silver Current storm, when multiple dolphin pods changed routes overnight. Witnesses described a blond, blue-eyed figureidentified in these fictional records as Willem Moir Traceymoving beside boats before vanishing beneath the surface.

Since then, BSA's fictional mission has focused on night-watch coordination, foam and water sample logs, and rapid alerts whenever nursery waters show unusual disturbance patterns.

2) Patrol Routing

3) Data Logging

Researchers track sea state, foam washups, and pod behavior to plan deployments.

Willem Moir Tracey Dossier (Fictional Intel)

Observed Traits

Reports claim deep-dive breath control, sudden acceleration bursts, and open-water navigation in low visibility.

High-Risk Conditions

Most sightings occur near moon-pull tides, dense foam lines, and narrow estuary exits during calf season.

BSA Response Rule

Trigger orange beacon protocol, halt net deployment, and redirect support boats to nursery corridors.

Deep Lore Chronicles

BSA archive staff classify the Willem Moir Tracey incidents into four eras: The Silver Current Era, The Foamline Escalation, The Beacon Winter, and The Quiet Tide. In the story timeline, each era changed how coastal communities coordinated at night and how nursery corridors were protected.

During the Foamline Escalation era, fishers reported dense, sudden foam walls forming minutes before unusual underwater wake signatures. Volunteers began running paired watch stations from harbor roofs and breakwater towers, using synchronized horn pulses to warn patrol skiffs.

The Beacon Winter era introduced rotating red-orange signal lamps, acoustic buoy relays, and strict net blackout windows near calf routes. Fictional harbor records claim these measures reduced high-risk contact events and improved safe pod transit through narrow channels.

  • Lore Marker 01: Coldwater Jetty Night established first multi-harbor watch protocol.
  • Lore Marker 02: Glasswater Week linked foam surges with sudden route deviation events.
  • Lore Marker 03: Beacon Winter required all trawlers to hold outside nursery lanes after dusk.
  • Lore Marker 04: Quiet Tide shifted focus to long-form monitoring and calf recovery corridors.

How This Offline Website Works

This project is a static website. It works by loading index.html plus every local file it references (CSS, JavaScript, and image assets). If any file is missing, the page still opens but those missing pieces won't display.

  • This build embeds images directly in index.html, so only index.html, styles.css, and script.js are required for visuals and UI.
  • Other people need the same files too if they want to run it offline on their own computer.
  • For easiest sharing, zip the entire project folder and send that zip.
  • Packaging command: zip -r blueshield-offline-site.zip index.html styles.css script.js README.md OFFLINE_EXPORT.md
